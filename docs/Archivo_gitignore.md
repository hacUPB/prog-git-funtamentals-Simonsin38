## **¿Qué es?**

- Un archivo .gitignore es un archivo de configuración que se usa en proyectos que utilizan Git (control de versiones) para decirle qué archivos o carpetas NO deben ser rastreados ni subidos al repositorio.

## **¿Para qué sirve?**

- Sirve para evitar que Git incluya archivos innecesarios, temporales o sensibles en el historial del proyecto.
