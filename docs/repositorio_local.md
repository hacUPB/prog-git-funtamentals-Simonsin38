## - Crear carpeta
Para crear un repositorio primero necesitamos una carpeta para el repositorio, para crear la carpeta usamos: mkdir (Nombre del repositorio) 

## - Entrar en la carpeta
Para ubicarnos en la carpeta usamos cd (Nombre del repositorio)/

## - Iniciar repositorio local
Usaremos el comando "git init" para convertir la carpeta en un repositorio.
