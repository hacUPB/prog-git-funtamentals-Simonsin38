## - Ingresar a github
Buscamos github en el buscador y creamos una cuenta.
## - Crear repositorio
Le damos click a crear un nuevo repositorio, le ponemos un nombre y lo creamos.
## - Comandos
En nuestro repositorio local, utilizamos el comando touch README.md para crear un archivo de texto, lo editamos y posteriormente usamos "git add README.md" para pasarlo al Stage, luego "git commit -m "(Acción)", después usamos "git branch -M main" seguido de "git remote add origin (link del repositorio de github)" y finalmente usamos "git push -u origin main" para subir el repositorio local al remoto.
